## About DataJoint

**DataJoint** implements the [Relational Workflow Model](explanation/relational-workflow-model.md)â€”a data model where your database schema defines an executable data pipeline. Tables represent workflow steps, foreign keys encode dependencies, and the system handles job management, parallel execution, and provenance tracking. [Object storage](explanation/data-pipelines.md#object-augmented-schemas) integration enables seamless handling of large scientific data.
